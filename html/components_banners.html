Banners

Purpose/Usage

The top navigational banners are used directly on top of global navigations to indicate account upgrades and other settings. This is the primary location to upgrade and turn on two-factor authentication.

Upgrade Banner

The upgrade banner is applied above the golbal navigation for free accounts and stays there until an upgrade is completed. The user cannot clear this banner.

Two-Factor Authentication Banner

When two-factor authentication is turned off in the account this banner will display above the global navigaiton encouranging the user to enable it for for better security.

Guidelines

Placement

  • Fixed to the top of the page above the primary navigation.
  • Full screen width, 56px tall.
  • Grey background (#494C51)
    with a repeating three stripe pattern.
  • White text instructions with yellow or green call to action text and an added arrow for affordance.
  • Instructions and text is placed on the left side, right side is reserved for passive actions like a phone number.

Typography

Font-Family: ASAP
Font-Size: 14px
Instructional color: Bumble (#FFF)

Weight: Regular (400)
CTA color: Sandstorm (#F1D500)
or Canopy Green (#00BF4B)

Weight: Bold (700)

Content

The content of the banners should:
  • Be direct and clearly explain why the banner exists.
  • Have friendly call to actions, not overbearing or forceful. These banners have the potential to exist through out the entire users experience with Canopy. (Exp. Free Transcripts account)

Best Practices

  • Navigational banner should only be used for communicating main account settings options.
  • Two banners should never be used at the same time.
  • Banners should not be used to communicate application or user status.
  • The only way to clear a banner is by using the CTA to update account settings, banners should never be temporarily cleared using a cancel or ‘X’.
  • Do not make banners taller than 56px to preserve space.
  • Do not make banners a different color or use big buttons that will distract the user from actions inside the app.
  • The banner is designed to be subtle, not very obtrusive, so that a user may continue to use the app without ever removing it from view (free transcripts account)
