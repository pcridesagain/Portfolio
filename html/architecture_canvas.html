Canvas

Every great piece of art starts with a canvas.

Great interfaces feel like extensions of human interaction. We want Canopy to be simple, efficient and delightful, to accomplish this present content and information to the user in a natural and intuitive feel.

Using principles of depth and hierarchy, we have created a more tactile and intuitive interface that mimics the world we know and respect - reading a book or tapping an elevator button.

The Canvas

Every element that is presented to the user is laid out overtop of the canvas. This is simply a base layer that ensures that the elements that are laid on top have a strong content hierarchy and contrast. It draws the users eye to the most important elements of the application.
