Wells

There are four types/colors of wells to help separate information: Rare, Medium, Well Done and Charred (only used inside of a DT card.)

Guidelines

  • Should be used to separate or group like content within a card
  • Should only be used on cards
  • Should never be directly placed on the canvas
  • Should never be used in navigation items
  • Wells can be collapsed or expanded with another action/li>

Sizing

  • Corner radius: 5px
  • Horizontal + Vertical Padding: 24px
  • Margin: determined by surrounding elements

Best Practices

  • Vertically stacked wells should have a 24px margin between them
  • Wells can be used within other wells depending on size
  • Wells can be used as headers for cards, without margin and adapting the border radius of the card
